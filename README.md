# 🌍 ScanFood AI

AI-powered food scanning web app that analyzes products and detects whether food is HALAL, SHUBHALI, or HARAM.

---

## 🚀 Live Demo
👉 https://scanfood-ai.netlify.app

---

## 💡 Features

- 📷 Barcode / QR scanner (demo version)
- ⌨️ Manual code input
- 🧠 AI-based food status detection (simulation)
- 🌍 Random country detection
- 🇺🇿🇹🇷🇸🇦🇺🇸🇬🇧🇷🇺🇨🇳 country flags support
- 🟢 HALAL / 🟠 SHUBHALI / 🔴 HARAM results
- 📱 Fully responsive mobile UI

---

## ⚙️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Netlify Hosting

---

## 📱 How it works

1. Enter barcode or QR code
2. Click "Check"
3. System analyzes product (demo AI logic)
4. Shows:
   - Food status
   - Country
   - Product code
